# Cultos-Inomin-veis
Espaço para edição de ficha para RPG Cultos Inomináveis.
Space for editing an RPG sheet for Nameless Cults.

Hi everyone!
Nameless Cults is an RPG recently release in Brazil with Catarse crownfunding. Its base is the HITOS sistem. 
In Roll20 there are only 2 sheets that may be used to play, both in Spanish. One for general HITOS sistem, and the other not good for my taste.
So, I used the code from general HITOS sheet as base for a new sheet closer to the one in the rulebook. 
I would like to know if its possible to set it free for the whole community in Roll20 to use it and play Nameless Cults.
Thanks for your attention.
